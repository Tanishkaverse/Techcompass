# Techcompass
TechCompass is a student-focused career guidance platform prototype. Created from my own idea and vision, and developed using AI-assisted tools such as Claude AI and Replit while learning Python and software development.

TechCompass

Overview

TechCompass is a platform concept designed to help students explore technology careers, discover learning opportunities, and find the right path in the technology industry.

This repository contains the prototype version of TechCompass.

Important Note

I recently started my programming journey and am currently learning Python fundamentals.

The idea, vision, problem statement, and overall concept behind TechCompass were created by me. However, the prototype was developed with the assistance of AI-powered tools, primarily Claude AI and Replit.

This project was created as an experiment to understand how AI-assisted development can help transform ideas into working product prototypes.

Project Vision

Many students are interested in technology but often struggle to answer questions such as:

- Which technology field should I choose?
- What skills should I learn?
- What learning resources are available?
- What career opportunities exist in different tech domains?

TechCompass aims to provide a centralized platform that helps students navigate these questions.
